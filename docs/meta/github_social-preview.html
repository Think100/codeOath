Social Preview Image Source

This file generates the GitHub social preview card (shown when someone shares the repo link).

  1. The red dashed border shows the exact 1280x640px boundary
  2. Take a screenshot INSIDE the red border (do not include the border)
  3. Save as PNG
  4. Upload to GitHub: repo Settings > General > Social Preview > Edit > Upload

To change text or styling, edit this file and take a new screenshot. Fonts load from Google Fonts (needs internet).
