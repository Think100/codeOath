Welcome to this blog. This is the first post.

I write posts in Markdown and they are rendered as HTML pages. Each post lives in its own `.md` file inside the `posts/` directory.

## How it works

1. Write a new `.md` file in `posts/`
2. Add YAML front matter at the top (title, date, optional tags)
3. Write your content below the second `---`
4. Restart the server (or it reloads automatically in debug mode)

That is it.
