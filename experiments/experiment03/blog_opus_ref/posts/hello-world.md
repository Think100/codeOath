Welcome to my new blog! This is the very first post.

I built this blog using **Python**, **Flask**, and **Markdown**. Posts are written as plain `.md` files with a small front matter header for metadata.

## How it works

1. Write a Markdown file in the `posts/` directory
2. Add a front matter block with title, date, and optional tags
3. Start the Flask server
4. Your post appears on the main page

It really is that simple.
