Welcome to my blog! This is the very first post.

I built this blog with **Python** and **Flask**, rendering Markdown files into HTML pages. Each post is a simple `.md` file with a front-matter header.

## How It Works

1. Write a Markdown file in the `posts/` directory
2. Add a front-matter block with title, date, and optional tags
3. Start the server and your post appears automatically

Pretty simple, right?
